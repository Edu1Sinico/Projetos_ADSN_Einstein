Passo 1: Criar o diretório
Crie a pasta em C:\ chamada EinsteinGestaoAcademica


Passo 2: Criar a solução
Abra o prompt de comando na pasta criada acima e crie a solução .netcore com o seguinte comando e em seguida dê um "enter":

dotnet new sln -n "EinsteinGestaoAcademica.Backend"

Passo 3: Criar o projeto WebAPI

Agora rode o comando para criar o projeto WebAPI e em seguida dê um "enter":

dotnet new webapi -n "EinsteinGestaoAcademica.API"

Passo 4: Adicionar o projeto na Solução
Agora rode o comando para criar o projeto WebAPI e em seguida dê um "enter":

dotnet sln add EinsteinGestaoAcademica.API


Passo 5: Instalação do Scalar
Navege até a pasta C:\EinsteinGestaoAcademica\EinsteinGestaoAcademica.API e rode o comando a seguir para instalar o componente visual da nossa WebAPI:

dotnet add package Scalar.AspNetCore --version 1.2.*


Passo 6: Código no Program.cs
using Scalar.AspNetCore;

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddControllers();

// Add services to the container.
// Learn more about configuring OpenAPI at https://aka.ms/aspnet/openapi
builder.Services.AddOpenApi();
builder.Services.AddEndpointsApiExplorer();


var app = builder.Build();


app.MapOpenApi();
app.MapScalarApiReference(options => 
    {
        options.WithTitle("Einstein Gestão Acadêmica WebAPI")
               .WithTheme(ScalarTheme.DeepSpace)
               .WithDefaultHttpClient(ScalarTarget.CSharp, ScalarClient.HttpClient);
    });

app.MapControllers();

app.Run();
